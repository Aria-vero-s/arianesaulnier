Project 1: Modus Design

Modus Design is a dynamic e-commerce platform aimed at providing users with a versatile selection of pre-made web templates and custom design services.

Technologies used: HTML, CSS, JavaScript, Django(Python), Bootstrap, ElephantSQL, Heroku, AWS3, Stripe.



At its core, Modus Design aims to create a user-friendly environment for both customers and administrators, facilitating effective engagement with web templates and design services.

Project Overview

  • User Experience (UX): Users can easily fill out a form to receive an automatic quote tailored to their needs. Seamless navigation and access to social media links. User registration enables the purchase of graphical designs, and subscribing to the newsletter keeps users informed of updates and changes.
  • Site Owner goals: The site owner can efficiently manage orders, showcase their work, and utilize CRUD functionality for adding, editing, or deleting products, artists, and classes.
  • Payment System: A secure payment system, powered by Stripe, enables users to conveniently purchase HTML templates or hire design services.
  • Products and Services: Modus Design offers a diverse product catalog featuring web design templates categorized into Website Design and Graphic Design.

Tools and Technologies

  • Front-End: HTML5, CSS3, JavaScript
  • Back-End: Django (Python), AllAuth (Django Packages)
  • Librairies: FontAwesome, Bootstrap 4.4.1
  • Database Services: ElephantSQL (PostgreSQL)
  • Object Storage Services: AWS S3
  • Deployment : Heroku
  • Version Control: VisualCodeStudio, Git/GitHub
  • Others: Stripe (Payment System)

Project 3: Art Gallery

The ArtPulse Django project is a web platform that offers users a unique space to explore, share, and connect with other artists. Developed using the Django framework in Python, the website encompasses a variety of features, such as artwork search, user account management, artwork publication, and interactive features such as likes and comments.

Technologies used: Django, Python, HTML, CSS, JavaScript, AWS3, Bootstrap, ElephantSQL, Heroku.



Project Overview

  • Efficient Artwork Discovery: Users can employ various filters, including title, artist name, keywords, and category, ensuring a tailored and user-friendly exploration experience.
  • Database Management: The system ensures that users can not only publish their artworks but also manage them effectively. This includes updating or deleting artworks from their profiles, fostering a sense of ownership and control for artists over their portfolio.
  • Inspired UI Design and User Interaction: The design is complemented by features that encourage interaction, such as full-screen image views, commenting, and rating. The platform's ethos, highlighted on the homepage, underscores the core values of connecting artists, facilitating artwork sharing, and promoting the discovery of new artistic endeavors and individuals.

Tools and Technologies

  • Front-End: HTML5, CSS3, JavaScript, Bootstrap
  • Back-End: Django (Python), AllAuth (Django Packages)
  • Database Services: ElephantSQL (PostgreSQL)

Project 5: Python Terminal

Enviro is a Python terminal survey designed to gather user responses on the topic of climate change. Developed to run on the Code Institute mock terminal deployed on Heroku, this project provides a simple yet effective platform for users to express their views on climate-related questions.

Technologies used: Python, Google Sheet API, Heroku.



Project Overview

  • Input Validation and Error-Checking: Implements input validation to catch errors, providing feedback and allowing users to correct their responses.
  • Data to Google Sheet: Utilizes the Google Sheet API to send user responses to a linked Google Sheet for analysis.
  • Data model: Utilizes a Python list to hold questions and multiple-choice answers. A for loop collects answers and verifies them, with responses sent to the Google Sheet using the SHEET variable and Google Sheet API.

  • APIs: Google Sheet API
