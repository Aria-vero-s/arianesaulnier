Project 1: CafeZest

This project aims to create a modern and user-friendly online platform for CafeZest, a fictional cafe. The primary objectives of this project were to establish a strong online presence, create a booking functionality, and provide an enjoyable digital experience for restaurant patrons.



Project Overview

  • Establish Online Presence: Create a modern and attractive website to showcase the cafe's menu, ambiance, and location.
  • Booking Functionality: Implement a reservation system that allows customers to book tables online.
  • Enhanced User Experience: Ensure a seamless and enjoyable experience for website visitors.

Tools and Technologies

  • Front-End: HTML5, CSS3, JavaScript, Django
  • Third-party APIs: Google Maps API, Email Service API
  • Version Control: Git/GitHub

Project 2: Pianist Website

I'm passionate about creating beautiful websites that not only look good but also function seamlessly. This single-page website showcases a pianist in an elegant and functional design.



Project Overview

  • Project Name: Pianist Website

The Pianist Website project showcases my ability to meet specific client requirements and to customize websites according to individual preferences. I approach each project with a problem-solving mindset, ensuring that the end result is a well-crafted and fully functional website. This also served as a valuable opportunity for me to hone my web development skills.

Working with a client

  • Client Collaboration and Problem Solving: I customized the website based on client requests. However, while open to client ideas, I exercised professional judgment on design elements maintaining functionality and integrity.
  • Multilingual Support: The client wanted the inclusion of translation buttons for English, French, and Italian. I used JavaScript in order to achieve this and stored the translation texts in JavaScript objects.
  • Responsive Design: The website is fully responsive, adapting to different screen sizes, showcasing my skills in creating websites that are accessible on various devices.
  • User-Centric Approach: I made the user experience a top priority, with a focus on both aesthetics and functionality.

Tools and Technologies

  • Front-End: HTML5, CSS3, JavaScript
  • Version Control: VisualCodeStudio, GitHub
